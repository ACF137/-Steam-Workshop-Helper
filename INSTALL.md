# Installation Guide

1. Open your browser
2. Go to the extensions page:
   - Chrome → `chrome://extensions/`
   - Edge → `edge://extensions/`
   - Opera GX → `opera://extensions/`
3. Enable **Developer mode**
   - Chrome / Opera GX → toggle in the **top-right corner**
   - Edge → toggle at the **bottom of the left sidebar**
4. Click **"Load unpacked"** (Opera GX: **"Load unpacked extension"**)
5. Find and select the **`dist`** folder — not a ZIP file, the folder itself
6. Done — the extension is ready to use

> ⚠️ Don't delete or move the `dist` folder after loading — the browser reads it directly.

---

## Verify it works

Open any Steam Workshop upload page:
```
https://steamcommunity.com/sharedfiles/edititem/767/3/
```
The **SteamKit** panel should appear in the top-left corner automatically.  
If the page was already open — refresh it first (F5).

---

## Updating

1. Replace the contents of your `dist` folder with the new version
2. Go to the extensions page and click 🔄 next to **Steam Workshop Helper**

---

## Uninstalling

Go to the extensions page → find **Steam Workshop Helper** → click **Remove**.
